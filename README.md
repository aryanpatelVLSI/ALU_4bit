# 4-Bit ALU in Verilog HDL

## Overview

JOURNEY BEGINS!! This is my first Verilog HDL project published on GitHub. The goal of this project was to learn digital design fundamentals, simulation and Git/GitHub workflow while using Xilinx Vivado.

This project implements a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL in Xilinx Vivado. The ALU performs arithmetic, logical, and shift operations based on a 3-bit selection input (SEL). The design was verified through behavioral simulation using a dedicated testbench.

## Supported Operations

- Addition (A + B)
- Subtraction (A - B)
- AND
- OR
- XOR
- NOT
- Left Shift
- Right Shift
